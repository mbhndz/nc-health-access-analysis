NC County Health Burden and Care Access Analysis



This is a small exploratory analysis of county-level health burden and physician access in North Carolina using public health data.



The project is mainly intended to practice and demonstrate basic data cleaning, exploratory analysis, and written interpretation with a real dataset.



In the notebook all of the following is done:



\- Load and inspect the county health data.

\- Clean selected variables for analysis.

\- Compare counties by health burden and physician access.

\- Identify counties with health burden and physician access both above median levels.

\- Compare this group with statewide county medians on a few related variables.
